# Index Google

Index wrapper for Google Gmail and Calendar libraries. PSR-4 :)

# Gmail

Get emails, send email, hooks - pub/sub

# Calendar

Get calendar events, hooks - pub/sub

## Install

``` bash
composer require index-io/google
```

## Usage

``` php
$caledar = new Index\Google\Calendar();
```

## Testing

``` bash
$ vendor/bin/phpunit
```
